Getting Started

What is AutoArr?

AutoArr is an intelligent orchestration layer that sits between you and your media automation stack. It uses AI to understand your natural language requests, optimize your service configurations, and automatically recover from download failures.

Key Benefits:
  • Natural language content requests ("Download The Bear Season 3")
  • AI-powered configuration auditing and optimization
  • Automatic download failure recovery
  • Real-time monitoring across all services

System Requirements

  • 💻
    Docker Host

    Any system running Docker (Linux, Windows, macOS, Synology NAS)

  • 🔒
    API Keys

    API keys for SABnzbd, Sonarr, Radarr (Plex optional)

  • 🤖
    LLM API Key

    Claude API key or compatible LLM for AI features

Quick Start

# Pull and run AutoArr
docker run -d \
  --name autoarr \
  -p 8088:8088 \
  -e SABNZBD_URL=http://your-sabnzbd:8080 \
  -e SABNZBD_API_KEY=your-api-key \
  -e SONARR_URL=http://your-sonarr:8989 \
  -e SONARR_API_KEY=your-api-key \
  -e RADARR_URL=http://your-radarr:7878 \
  -e RADARR_API_KEY=your-api-key \
  -e CLAUDE_API_KEY=your-claude-key \
  autoarr/autoarr:latest

Then open http://localhost:8088 in your browser.

Connecting Services

Supported Services

Plex

Media server for streaming (optional)

Finding Your API Keys

  1. Open SABnzbd web interface
  2. Go to Config > General
  3. Look for "API Key" under Security section
  4. Click "Show" to reveal and copy the key
  1. Open Sonarr web interface
  2. Go to Settings > General
  3. Under "Security", find the API Key
  4. Copy the key
  1. Open Radarr web interface
  2. Go to Settings > General
  3. Under "Security", find the API Key
  4. Copy the key
  1. Sign in to Plex Web App
  2. Browse to any library item
  3. Click "Get Info" or view XML
  4. Look for X-Plex-Token in the URL

Or use: Plex's official guide

Settings Page

After first launch, navigate to the Settings page to configure your services. Each service needs:

  • URL - The address of your service (e.g., http://192.168.1.100:8989)
  • API Key - Your service's API key
Tip: AutoArr will test the connection when you save settings and show you the status of each service.

Chat Interface

Natural Language Requests

The Chat interface lets you interact with your media stack using natural language. AutoArr's AI understands what you want and routes requests to the appropriate service.

Download The Bear season 3
Sonarr AutoArr will search Sonarr for "The Bear", add it if not present, and ensure Season 3 is monitored.
Get Dune Part Two in 4K
Radarr AutoArr will search Radarr for "Dune Part Two" and configure it to download in your best quality profile.
What's downloading right now?
SABnzbd AutoArr will check the SABnzbd queue and report current downloads with progress and ETA.
Why did my last download fail?
SABnzbd AutoArr will check SABnzbd history, find the failed item, and explain the failure reason.

Service Health

Understanding Health Scores

The Service Health page provides a comprehensive assessment of your media stack's configuration. Each service receives a health score from 0-100 based on various checks.

90-100 Excellent - Optimal configuration
70-89 Good - Minor improvements possible
50-69 Warning - Some issues need attention
0-49 Critical - Significant problems detected

What Gets Checked

SABnzbd Checks

  • Server connections and SSL
  • Article cache size
  • Connection count
  • Disk space availability
  • Warning/error logs
  • Direct unpack settings

Sonarr Checks

  • Indexer configuration
  • Download client setup
  • Root folder disk space
  • Built-in health checks
  • Quality profile setup
  • Unmonitored series ratio

Radarr Checks

  • Indexer configuration
  • Download client setup
  • Root folder disk space
  • Built-in health checks
  • Quality profile setup
  • Movies without files

Plex Checks

  • Library configuration
  • Movie/TV library presence
  • Active streaming sessions
  • Transcoding detection
  • Library item counts

Auto-Fix Features

Some issues can be automatically fixed by AutoArr. Look for the Auto-Fix button next to fixable issues.

Example: If SABnzbd has too many warnings logged, AutoArr can clear them for you with one click.
