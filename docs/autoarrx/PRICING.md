1. **AutoArr is 100% free forever** - No limitations, no nags
2. **Premium is optional** - AutoArr works great standalone
3. **Privacy-first** - No media file access, only metadata

5. **Fair value** - Premium features worth paying for

## Frequently Asked Questions

### Will AutoArr stay free?

**Yes.** AutoArr is GPL-3.0 open source and will always be 100% free. Premium features are entirely optional add-ons.

### Can I self-host AutoArrX?

No. AutoArrX is a cloud service (SaaS) due to:
- Premium LLM costs
- Cloud infrastructure
- Ongoing maintenance

However, AutoArr itself is self-hosted and works perfectly without AutoArrX.

### What happens if I cancel Premium?

- Your AutoArr instance continues working normally
- Cloud features stop (notifications, analytics)
- Local features remain fully functional
- You can re-subscribe anytime

### Is there a trial period?

Yes. 14-day free trial of Premium features (planned).

### Do you offer discounts?

- Students: 50% off (with .edu email)
- Open source contributors: Contact us
- Annual plans: 17% discount

### Can I switch tiers?

Yes, anytime. Upgrades are immediate, downgrades take effect at renewal.
