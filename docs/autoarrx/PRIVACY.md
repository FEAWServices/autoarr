# AutoArrX Privacy Policy

**Status:** Pre-launch draft
**Last Updated:** 2025-01-12

## Our Privacy Commitment

AutoArrX is designed with **privacy-first** principles. We believe you should have complete control over your data and media library.

## What We Collect

### Required Data (for Premium features)

1. **Account Information**
   - Email address
   - Password (hashed, never stored in plaintext)
   - Payment information (via Stripe, we never see card numbers)

2. **Configuration Metadata**
   - Service configurations (Sonarr, Radarr, SABnzbd URLs)
   - Quality profiles
   - Download client settings
   - Notification preferences

3. **Activity Logs**
   - Download attempts
   - Configuration audits
   - Content requests
   - System events

4. **Library Statistics**
   - Number of movies/TV shows
   - Storage usage (total size)
   - Download success rates
   - Quality distribution

### What We NEVER Collect

We **explicitly do not collect**:

- ❌ Media file contents
- ❌ File paths or names
- ❌ Personal information beyond email
- ❌ Usenet credentials or API keys
- ❌ Indexer API keys
- ❌ Torrent trackers
- ❌ IP addresses of your media servers
- ❌ Browsing history
- ❌ Social media data

## How We Use Your Data

### Cloud Intelligence
- LLM analysis of configurations (anonymized)
- Recommendation generation
- Download failure analysis

### Smart Notifications
- Send notifications to your configured channels
- Track notification delivery status

### Predictive Analytics
- Aggregate statistics (anonymized)
- Trend analysis
- Failure prediction models

### Service Improvement
- Bug reports and error logs
- Feature usage statistics
- Performance metrics

## Data Storage & Security

### Storage Location
- **Primary:** Azure (US East)
- **Backup:** Azure (US West)
- **EU users:** EU data residency (planned Q2 2025)

### Security Measures

1. **Encryption**
   - End-to-end encryption for all data in transit (TLS 1.3)
   - Encryption at rest (AES-256)
   - Database encryption (Azure SQL TDE)

2. **Access Control**
   - Multi-factor authentication (MFA)
   - Role-based access control (RBAC)
   - Principle of least privilege

3. **Monitoring**
   - 24/7 security monitoring
   - Automated threat detection
   - Regular security audits

4. **Compliance**
   - SOC 2 Type II (planned)
   - GDPR compliant
   - CCPA compliant

## Data Retention

| Data Type | Retention Period |
|-----------|------------------|
| Account information | Until account deletion |
| Activity logs | 90 days |
| Configuration backups | 30 days |
| Analytics data (anonymized) | 2 years |
| Payment records | 7 years (legal requirement) |

## Data Sharing

### We DO Share With:

1. **Service Providers**
   - Azure (hosting)
   - Stripe (payments)
   - SendGrid (email)
   - Analytics providers (anonymized data only)

2. **Legal Requirements**
   - Law enforcement (with valid warrant)
   - Court orders
   - Legal compliance

### We NEVER Share:
- ❌ Your data with advertisers
- ❌ Your data with data brokers
- ❌ Your media library information
- ❌ Your personal information for marketing

## Your Rights (GDPR)

You have the right to:

1. **Access** - Request a copy of your data
2. **Rectification** - Correct inaccurate data
3. **Erasure** - Delete your data (right to be forgotten)
4. **Portability** - Export your data in machine-readable format
5. **Restriction** - Limit how we process your data
6. **Objection** - Object to certain data processing

To exercise these rights: privacy@autoarr.io

## Data Deletion

### What happens when you delete your account:

**Immediate:**
- Account disabled
- Access revoked
- Cloud features stop

**Within 30 days:**
- Account data deleted
- Activity logs deleted
- Configuration backups deleted
- Notification history deleted

**Retained (anonymized):**
- Aggregate statistics
- Anonymized analytics
- Payment records (legal requirement)

## AutoArr (Free) vs AutoArrX (Premium)

### AutoArr (GPL)
- **100% local** - No cloud connection
- **No data collection** - Everything stays on your server
- **No accounts** - No registration required
- **Zero tracking** - Completely private

### AutoArrX (Premium)
- **Optional cloud** - You choose what to sync
- **Minimal data** - Only what's needed for features
- **Transparent** - This policy explains everything
- **Secure** - Enterprise-grade security

**Bottom line:** AutoArr is completely private. AutoArrX is privacy-conscious with clear data practices.
